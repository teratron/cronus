# Office View (projection)

Live picture of this office, derived from roster + board + activity (not a source of truth).
Two render modes over one model: interaction graph (nodes=agents/tasks, edges=reporting/
collaboration/assignment) and a spatial floor skin (rooms/seats).

- `layout.json` — cosmetic placement only (positions/rooms); deleting it re-lays-out from the model.

The home workspace additionally shows a building overview of all offices (floors) + the building boss.
Observational (the client watches; the office does the work).
